C++ windows form application, similar to students-notebook, which has the following
functionality:
1. Connects to local DB and displays in the form of a table the content
of the DB, which is information about book authors.
2. When the user clicks on an author, a second table appears, which gets
information from the database about the books(name, genre, ISBN, etc) from that author.
3. The user is allowed to add, delete and edit both authors and their books.
